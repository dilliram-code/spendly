# Spendly 💰

Spendly is a personal expense tracking application built to help users manage their daily spending, monitor financial habits, and gain better control over their expenses. The project was developed with the assistance of Claude Code and focuses on providing a clean, simple, and user-friendly experience for tracking personal finances.

---

## 🚀 Features

- Add daily expenses easily
- Track spending history
- Categorize expenses
- View total spending
- Responsive and clean user interface
- Fast and lightweight application
- User-friendly design for personal finance management

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- SQLite / CSV Storage
- HTML & CSS (for styling)

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/spendly.git
```

### 2. Navigate to the Project Directory

```bash
cd spendly
```

### 3. Create a Virtual Environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 📦 Install Dependencies

```bash
pip install -r requirements.txt
```
